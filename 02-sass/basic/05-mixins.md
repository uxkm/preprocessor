# Sass 믹스인(Mixins) 활용 가이드 - 코드 재사용성 극대화

Sass에서 믹스인(Mixins)은 자주 사용되는 스타일 블록을 재사용할 수 있도록 해주는 기능으로, 코드의 중복을 줄이고 유지보수성을 높이는 데 매우 유용합니다. 이 문서에서는 Sass 믹스인을 정의하고 사용하는 방법을 소개합니다.

## Sass 믹스인이란? - 스타일 코드의 재사용

Sass 믹스인은 CSS 스타일을 함수처럼 정의하고, 필요할 때마다 호출하여 사용할 수 있습니다. 이를 통해 반복적인 스타일 작업을 간소화하고, 코드의 재사용성을 극대화할 수 있습니다.

### Sass 믹스인 정의와 사용

믹스인은 `@mixin` 지시어로 정의하며, `@include` 지시어로 호출합니다. 믹스인에 인자를 전달하여 동적으로 스타일을 생성할 수도 있습니다.

```
@mixin border-radius($radius) {
  -webkit-border-radius: $radius;
     -moz-border-radius: $radius;
          border-radius: $radius;
}

.box {
  @include border-radius(10px);
}
```

위 예제에서는 `border-radius` 믹스인을 정의하여, `.box` 클래스에 적용하고 있습니다. 이 믹스인은 여러 요소에서 재사용될 수 있습니다.

## Sass 믹스인의 고급 기능

### 믹스인에 인자 전달하기

Sass 믹스인은 인자를 받아 동적으로 스타일을 생성할 수 있습니다. 기본 인자 값도 설정할 수 있어, 유연한 스타일 작성이 가능합니다.

```
@mixin box-shadow($x: 0px, $y: 0px, $blur: 5px, $color: rgba(0, 0, 0, 0.5)) {
  -webkit-box-shadow: $x $y $blur $color;
          box-shadow: $x $y $blur $color;
}

.panel {
  @include box-shadow($x: 2px, $y: 4px);
}
```

위 예제에서는 기본 인자를 정의하여, 필요에 따라 특정 값을 재정의할 수 있습니다.

### 가변 인자를 사용한 믹스인

Sass에서는 믹스인에 가변 인자를 전달할 수 있습니다. 이를 통해 여러 개의 인자를 배열처럼 처리할 수 있습니다.

```
@mixin font-sizes($sizes...) {
  @each $size in $sizes {
    .font-#{$size} {
      font-size: $size;
    }
  }
}

@include font-sizes(12px, 14px, 16px);
```

위의 예제에서는 가변 인자를 사용하여 다양한 크기의 폰트 사이즈 클래스를 한 번에 생성할 수 있습니다.

## 믹스인과 상속의 차이점

Sass의 믹스인은 코드의 재사용성을 높이는 데 매우 유용하지만, 상속(`@extend`)과는 다릅니다. 믹스인은 스타일 블록을 복사하여 여러 선택자에서 사용할 수 있도록 해주고, 상속은 선택자 간의 관계를 형성합니다. 믹스인은 독립적인 코드 재사용이 필요할 때 사용하며, 상속은 공통 스타일을 공유할 때 유용합니다.
